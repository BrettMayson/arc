# ARC - Advanced Remote Control

A comprehensive UAV control system for Arma 3 that revolutionizes how players interact with unmanned vehicles. ARC transforms UAV gameplay with battery management, realistic signal simulation, and an advanced ISR system.

## Features

### Persistent Battery System

- Recoverable batteries when packing UAVs
- Battery level is retained when removing batteries from UAVs

### Deployable UAV Items

- UAVs are magazines that go in backpacks, not entire backpacks
- Extra backpack space for a few spare batteries

### Signal & Connectivity System

- **Line of Sight (LOS)** connection mode for direct control
- **Satellite (SAT)** connection mode for large vehicles
- Resolution of video depends on signal frequency
- Signal strength based on transmission power, distance, terrain, obstacles, and weather conditions

### Advanced ISR System

- Control UAV transmission settings (frequency, power)
- Snap to targets and grids, toggle markers, adjust units
- See ACE laser codes

### ACE3 Interaction

- Arm the Bustard with any weapon
- Arm the IED drone with any configuration
